# Portal Gerencial — Demo Comercial

Demo portable del **Portal Gerencial para Distribuidoras** de Vibras Positivas HM. Un solo archivo HTML con datos 100% ficticios ("Demo S.A.S.") que funciona sin internet, sin API y sin base de datos — ideal para presentaciones comerciales en PC o celular.

## ¿Qué muestra?

- **Dashboard gerencial**: ventas netas, devoluciones, total cartera, días cartera, stock crítico y alertas.
- **Cartera por edades**: barra apilada por rangos y detalle por cliente con semáforo de estado.
- **Análisis de vendedores**: ventas, cobros, cumplimiento de meta y tendencia.
- **Inventario real**: valorizado por categoría al costo ponderado, con KPIs de referencias bajo mínimo y sobre máximo.
- **Sin movimiento**: productos sin venta en 90+ días con capital inmovilizado y días sin rotación.
- **Sugerido de pedidos**: productos bajo stock mínimo con cantidad sugerida por proveedor.
- **Pareto de clientes**: clasificación 80/20 con porcentaje acumulado.
- **CXP Proveedores**: cuentas por pagar con vencimientos, alertas y descuentos por pronto pago.
- **Tesorería**: saldos bancarios (enmascarados), recaudo del día, pagos programados y flujo proyectado a 7 días.
- **Presupuesto de cobro**: meta mensual vs. cobrado real por vendedor, con proyección de cierre.

## Características

- ✅ Un solo archivo `.html` — se abre con doble clic o desde WhatsApp.
- ✅ 100% responsive: sidebar colapsable en móvil.
- ✅ Fecha de Colombia (UTC-5) en tiempo real.
- ✅ Insignia visible "DEMO — DATOS FICTICIOS" en todas las vistas.
- ✅ Sin conexión requerida: los datos están embebidos.

## Uso

1. Abrir `demo-portal-gerencial.html` en cualquier navegador (PC o celular).
2. Para publicar online: subir a GitHub Pages, Netlify u Hostinger.
3. Antes de compartir en redes: subir la imagen `og-demo-portal.png` (1200×630) a `https://vibraspositivashm.com/img/` para la vista previa de enlaces.

## Producto real

La versión comercial se conecta al ERP SQL Server del cliente (solo lectura) mediante API Node.js instalada en su propio servidor. Los datos nunca salen de la empresa.
